## La Misión
Fragments no es solo un repositorio de clases; es una **plataforma de aprendizaje gamificada** diseñada para transformar la educación técnica. Nació de la necesidad de centralizar material académico y evolucionó en un ecosistema donde aprender es una experiencia inmersiva, competitiva y, sobre todo, humana.

> *"En Fragments, no solo consumes contenido; forjas tu identidad como desarrollador junto a tu compañero digital."*

---

## Características Únicas

### El Efecto Protégé (Tu Mascota)
En Fragments **eres tu mascota**. Para consolidar lo aprendido, debes explicarle conceptos a tu mascota pixel-art.
- **Aprendizaje Activo**: Enseñar es la mejor forma de aprender.
- **IA Socrática**: Integración con Google Gemini para darte pistas, nunca las respuestas.
- **Identidad**: Mantente anónimo(a) ante tus pares, pero demuestra tu valía al profesor.

### El Grafo del Conocimiento
Visualiza tu progreso con un **mapa interactivo (estilo Obsidian)** de todos los temas.
- **Dependencias Reales**: No puedes saltar al nivel Pro sin dominar las bases.
- **Progreso Visual**: Mira cómo se ilumina tu camino a medida que completas microevaluaciones.

### El DIY (Do It Yourself)
Sección exclusiva para ex-alumnos y profesionales.
- **Proyectos Reales**: Enfréntate a desafíos de la industria sin guías paso a paso.
- **Radar de Habilidades**: Tu portfolio se construye solo, basado en la calidad de tus repos de GitHub y la disciplina de tus commits.

### Onboarding: Semana Cero
Tu entrenamiento comienza con tres misiones críticas:
1.  **Protocolo Gemini**: Genera y asegura tu propia API Key.
2.  **Disciplina Git**: Configura tu entorno y haz tu primer commit real.
3.  **Vínculo Digital**: Nombra y personaliza a tu mascota.

## Stack Tecnológico
Fragments está construido sobre la vanguardia del desarrollo web:

- **Frontend core**: Next.js 15 (App Router) + Tailwind CSS.
- **Backend & Auth**: Supabase (PostgreSQL).
- **Inteligencia**: Google Gemini API (IA Socrática).
- **Visualización**: D3.js para el Knowledge Graph.
- **Arte**: Pixel Art animado personalizado.

## Instalación (Entorno de Desarrollo)

Clona el núcleo y arranca el sistema:

```bash
# 1. Clonar el repositorio
git clone https://github.com/tu-usuario/fragments.git

# 2. Instalar dependencias
bun install

# 3. Configurar variables de entorno (.env.local)
# Copia el ejemplo y añade tus credenciales de Supabase
cp .env.example .env.local

# 4. Iniciar motor de desarrollo
bun run dev
```
